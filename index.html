Ticket Mangement System (Personal Project)

The Ticket Management System is a Laravel-based web app built with Blade, Breeze for authentication, Tailwind CSS for styling, and MySQL for data management. It supports three roles: Admin, Support, and User. Admins can manage users and tickets, Support can update tickets and give feedback, and Users can create and track tickets. The system integrates Mailtrap for email verification and sends notifications for all key actions to ensure smooth communication and efficient issue resolution.
